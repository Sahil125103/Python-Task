# Python Basics: Arithmetic & Greeting Program

This repository contains two beginner-level Python tasks:

- **Task 1:** Perform arithmetic operations (addition, subtraction, multiplication, and division) on two user-input numbers.
- **Task 2:** Greet the user by asking their first and last name.

---

## 🧮 Task 1: Arithmetic Calculator

### Description:
Prompts the user to input two numbers and performs:

- Addition
- Subtraction
- Multiplication
- Division (with zero-division check)

### Example:
Enter the first number: 10
Enter the second number: 2

Results:
Addition: 12.0
Subtraction: 8.0
Multiplication: 20.0
Division: 5.0


## 🙋 Task 2: Greeting Message

### Description:
Takes the user's first and last name as input and displays a personalized greeting.

### Example:
Enter your first name: Sahil 
Enter your last name: Kumar

Hello, Sahil Kumar! Welcome to the Python program.
