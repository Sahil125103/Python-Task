# SportsNews - Sports Players News Website

A modern, responsive website for sports players news with beautiful UI and interactive features.

## 🏆 Features

### Design & UI
- **Modern Design**: Clean, professional layout with gradient backgrounds
- **Responsive**: Fully responsive design that works on all devices
- **Smooth Animations**: CSS animations and transitions for enhanced UX
- **Beautiful Typography**: Inter font family for excellent readability
- **Interactive Elements**: Hover effects, click animations, and smooth scrolling

### Sections
- **Hero Section**: Eye-catching landing area with floating player cards
- **Latest News**: Grid layout with featured news articles
- **Top Players**: Player profiles with stats and ratings
- **Popular Teams**: Team information with achievements
- **About Section**: Website information with statistics
- **Footer**: Contact information and social links

### Interactive Features
- **Mobile Navigation**: Hamburger menu for mobile devices
- **Smooth Scrolling**: Navigation links scroll smoothly to sections
- **Active Navigation**: Highlights current section in navigation
- **News Ticker**: Rotating news headlines at the top
- **Back to Top Button**: Appears when scrolling down
- **Theme Toggle**: Dark/light mode switch (placeholder)
- **Search Functionality**: Search through news articles
- **Analytics Tracking**: Event tracking for user interactions

### Technical Features
- **Vanilla JavaScript**: No external dependencies
- **CSS Grid & Flexbox**: Modern layout techniques
- **Intersection Observer**: Scroll-based animations
- **Font Awesome Icons**: Professional iconography
- **Google Fonts**: Inter font family
- **Custom Scrollbar**: Styled scrollbar for better UX

## 🚀 Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge)
- No additional software required

### Installation
1. Clone or download the project files
2. Open `index.html` in your web browser
3. The website will load with all features enabled

### File Structure
```
sports-news-website/
├── index.html          # Main HTML file
├── styles.css          # CSS styles and animations
├── script.js           # JavaScript functionality
└── README.md          # This documentation
```

## 📱 Responsive Design

The website is fully responsive and optimized for:
- **Desktop**: 1200px+ (full layout with all features)
- **Tablet**: 768px - 1199px (adjusted grid layouts)
- **Mobile**: 480px - 767px (single column layouts)
- **Small Mobile**: <480px (optimized for small screens)

### Color Palette

## 🔧 Customization

### Adding New Content
1. **News Articles**: Add new `.news-card` elements in the news section
2. **Players**: Add new `.player-card` elements in the players section
3. **Teams**: Add new `.team-card` elements in the teams section

### Styling Changes
- **Colors**: Modify CSS custom properties in `styles.css`
- **Layout**: Adjust grid and flexbox properties
- **Animations**: Modify keyframes and transition properties

### JavaScript Features
- **Search**: Extend the `searchNews()` function
- **Analytics**: Implement real analytics in `trackEvent()`
- **Notifications**: Use `showNotification()` for user feedback
